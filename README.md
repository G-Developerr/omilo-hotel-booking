## ✨ Features
- 🔐 Multi-account login system
- 🏨 Hotel search & booking
- 📅 Booking management (view, cancel)
- ❤️ Favorites list
- 🌍 Multi-language support (EN, GR, ES, FR, DE, JP)
- 🌙 Dark / Light mode
- 💾 Bookings saved per account (localStorage)

## 🛠️ Tech Stack
- React
- Framer Motion
- React Icons
- OpenStreetMap

## ▶️ Run Locally
cd frontend
npm install
npm start
